Visualizador de fotos do Instagram

Esta aplicação foi desenvolvida para estudo da API do Instagram

Através dessa aplicação é possível realizar buscas de usuários do instagram e visualizar todas as suas fotos como se fosse um perfil da pessoa na rede social.


Fique a vontade para alterar o código. Caso seja de ajuda ou você utilize esta aplicação, peço que me mande um e-mail me informando.

É legal saber que estão utilizando algo que você desenvolveu :)
